## 🧩 학습 내용

- **소프트웨어 설계** : 요구사항 분석 / UI 설계  
- **소프트웨어 개발** : 서버 및 공통 모듈 구현  
- **데이터베이스 구축** : SQL 작성 및 최적화  
- **프로그래밍 언어 활용** : 알고리즘, 자료구조  
- **정보시스템 구축관리** : 보안, 네트워크, 패키징
